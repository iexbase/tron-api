## Install

```bash
> composer require wawilow/tron-api --ignore-platform-reqs
```

PS Install original library
```bash
> composer require iexbase/tron-api --ignore-platform-reqs
```
## Requirements

The following versions of PHP are supported by this version.

* PHP 7.4

## Testing
